# Scoot - A Custom WordPress Theme. 

### Description: 
Scoot is a fully responsive custom Wordpress theme for a scooter sharing company. The theme is built pixel perfect from a Figma design file.
The theme is visually pleasing with clean layout and beautiful typography. The UX is enhanced with subtle but appropriate animations of UI elements.

### Technology used:
HTML5, CSS3, PHP, JavaScript, WordPress
